## Wedding RSVP
  
▪ Send RSVPs online.
  
▪ Built using HTML, CSS, JavaScript, Node.js, and Express.js.
  
▪ A dynamic form makes the questions asked dependent on an invitee’s responses, e.g. if the invitee is attending on their own, then the names of guests attending with the invitee are not required.
  
▪ Responsive layout means the app can be resized across devices.
